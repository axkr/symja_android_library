## IdentityMatrix

```
IdentityMatrix(n)
```

> gives the identity matrix with `n` rows and columns.

```
IdentityMatrix({n,m})
```

> gives the identity matrix with `n` rows and `m` columns.

See
* [Wikipedia - Identity matrix](https://en.wikipedia.org/wiki/Identity_matrix)

### Examples

```
>> IdentityMatrix(3)
{{1, 0, 0}, {0, 1, 0}, {0, 0, 1}}
```
