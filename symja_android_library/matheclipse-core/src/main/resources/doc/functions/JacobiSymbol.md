## JacobiSymbol

```
JacobiSymbol(m, n)
```

> calculates the Jacobi symbol.

See
* [Wikipedia - Jacobi symbol](https://en.wikipedia.org/wiki/Jacobi_symbol)

### Examples

```
>> JacobiSymbol(1001, 9907)
-1
```
