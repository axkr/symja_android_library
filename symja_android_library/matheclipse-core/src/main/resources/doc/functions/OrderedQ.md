## OrderedQ

```
OrderedQ({a, b})
```

> is `True` if `a` sorts before `b` according to canonical ordering.

See
* [Wikipedia - Order theory](https://en.wikipedia.org/wiki/Order_theory)

### Examples

```
>> OrderedQ({a, b})
True

>> OrderedQ({b, a})
False
```
