## SSSTriangle

```
SSSTriangle(a, b, c)
```

> returns a triangle from 3 sides `a`, `b` and `c`.

See:
* [Wikipedia - Solution of triangles](https://en.wikipedia.org/wiki/Solution_of_triangles)
 

### Examples

``` 
>> SSSTriangle(10,10,10)
Triangle({{0,0},{10,0},{5,5*Sqrt(3)}})
```
