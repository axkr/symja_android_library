## CosIntegral

```
CosIntegral(expr)
```

> returns the cosine integral of `expr`.
  
See
* [Wikipedia - Trigonometric integral](https://en.wikipedia.org/wiki/Trigonometric_integral)

### Examples

```
>> CosIntegral(0)
-Infinity
```
