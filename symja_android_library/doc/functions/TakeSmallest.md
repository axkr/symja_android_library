## TakeSmallest

```
TakeSmallest({e_1, e_2, ..., e_i}, n) 
```

> returns the `n` smallest real values from the list `{e_1, e_2, ..., e_i}`.

### Examples

```
>> TakeSmallest(Prime(Range(10)), 3) 
{2,3,5}
```
