## CountDistinct
```
CountDistinct(list)
```
> returns the number of distinct entries in `list`.
 

### Examples

```
>> CountDistinct({3, 7, 10, 7, 5, 3, 7, 10})
4

>> CountDistinct({{a, a}, {a, a, a}, a, a}) 
3
```
