## Abs

```
Abs(expr)
```

> returns the absolute value of the real or complex number `expr`.
  

See:
* [Wikipedia - Absolute value](http://en.wikipedia.org/wiki/Absolute_value)
 

### Examples

```
>> Abs(-3)
3
```
