## RSolve

```
RSolve(equation, y(var), var)
```
> attempts to solve a recurrence `equation` for the function `y(var)` and variable `var`.

See:  
* [Wikipedia - Recurrence relation](https://en.wikipedia.org/wiki/Recurrence_relation)
