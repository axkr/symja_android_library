## Order

```
Order(a, b)
```

> is `0` if `a` equals `b`. Is `-1` or `1` according to canonical order of `a` and `b`.
 
See
* [Wikipedia - Order theory](https://en.wikipedia.org/wiki/Order_theory)

### Examples

```
>> Order(3,4)
1

>> Order(4,3)
-1
```
