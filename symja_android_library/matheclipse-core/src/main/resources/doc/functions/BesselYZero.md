## BesselYZero

```
BesselYZero(n, z) 
```

> is the `k`th zero of the `BesselY(n,z)` function. 

See
* [Wikipedia - Bessel function](https://en.wikipedia.org/wiki/Bessel_function)
* [Fungrim - Bessel functions](http://fungrim.org/topic/Bessel_functions/)

### Examples

```
>> BesselYZero(1.3, 3) 
9.031260842335175
```
