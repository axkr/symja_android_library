## WhittakerM

```
WhittakerM(a, b, z)
```

> returns the Whittaker function `M_a,b(z)`.

See
* [Wikipedia - Whittaker function](https://en.wikipedia.org/wiki/Whittaker_function)
 
### Examples

```
>> N(WhittakerM(2, 1/2, 2*2*I)) 
0.3080150149255876+I*(-8.938966760794024)
```
