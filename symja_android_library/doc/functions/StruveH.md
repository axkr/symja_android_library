## StruveH

```
StruveH(n, z)
```

> returns the Struve function `H_n(z)`.

See
* [Wikipedia - Struve function](https://en.wikipedia.org/wiki/Struve_function)
 
### Examples

```
>> StruveH(1.5, 3.5)
1.1319901169676305
```
