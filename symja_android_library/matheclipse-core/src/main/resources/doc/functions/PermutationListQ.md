## PermutationListQ

```
PermutationListQ(permutation-list)
```

> if `permutation-list` is a valid permutation list return `True`.
 
See 
* [Wikipedia - Permutation](https://en.wikipedia.org/wiki/Permutation)
	 
### Examples

```
>> PermutationListQ({5, 7, 6, 1, 3, 4, 2, 8})
True
```
