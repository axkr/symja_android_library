## CubeRoot

```
CubeRoot(n)
```

> finds the real-valued cube root of the given `n`.

### Examples

```
>> CubeRoot(16)
2*2^(1/3)

>> CubeRoot(-5)
-5^(1/3)

>> CubeRoot(-510000)
-10*510^(1/3)
 
>> CubeRoot(-5.1)
-1.7213006207263157

>> CubeRoot(b)
b^(1/3)   

>> CubeRoot(-0.5)
-0.793701
```
