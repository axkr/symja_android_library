## RemoveDiacritics

```
RemoveDiacritics("string")
```

> returns a version of `string` with all diacritics removed.
 

See:
* [Wikipedia - Diacritic](https://en.wikipedia.org/wiki/Diacritic)
 

### Examples

```
>> RemoveDiacritics("éèáàâ")
eeaaa
```
