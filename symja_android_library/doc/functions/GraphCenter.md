## GraphCenter

``` 
GraphCenter(graph)
```

> compute the `graph` center. The center of a `graph` is the set of vertices of graph eccentricity equal to the `graph` radius.
 
See:
* [Wikipedia - Graph center](https://en.wikipedia.org/wiki/Graph_center)
