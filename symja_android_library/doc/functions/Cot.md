## Cot

```
Cot(expr)
```

> the cotangent function.


See:
* [Wikipedia - Trigonometric functions](https://en.wikipedia.org/wiki/Trigonometric_functions)

### Examples
```
>> Cot(Pi/4)
1

>> Cot(0)    
ComplexInfinity 
 
>> Cot(1.)    
0.6420926159343308 
```
