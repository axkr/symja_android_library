## Erfi

```
Erfi(z)
```

> returns the imaginary error function of `z`.

See
* [Wikipedia - Error function](https://en.wikipedia.org/wiki/Error_function)
* [Fungrim - Error functions](http://fungrim.org/topic/Error_functions/)

### Examples


```
>> Erfi(-42*I*x) 
-I*Erf(42*x)
```
