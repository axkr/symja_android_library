## Polygon

```
Polygon({point_1, point_2 ...})
```

> represents the filled polygon primitive.

```
Polygon({{p_11, p_12, ...}, {p_21, p_22, ...}, ...})
```

> represents a number of filled polygon primitives.
 
### Examples

```
>> Graphics3D(Polygon({{0,0,0},{0,1,1},{1,0,0}}))
 -Graphics3D-
```
