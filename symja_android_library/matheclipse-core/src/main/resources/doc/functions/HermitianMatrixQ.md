## HermitianMatrixQ

```
HermitianMatrixQ(m)
```

> returns `True` if `m` is a hermitian matrix.

See 
* [Wikipedia - Hermitian matrix](https://en.wikipedia.org/wiki/Hermitian_matrix)
 

### Examples

```
>> HermitianMatrixQ({{2, 2 + I, 4}, {2-I, 3, I}, {4, -I, 1}})
True
```
