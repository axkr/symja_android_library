## MedianFilter

```
MedianFilter(list, r)
```

> filter which evaluates the `Median` of `list` for the radius `r`. 

See
* [Wikipedia - Median filter](https://en.wikipedia.org/wiki/Median_filter)
 

### Examples

```
>> MedianFilter({2,3,80,6}, 1) 
{5/2,3,6,43}
```
