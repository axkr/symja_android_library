## CatalanNumber

```
CatalanNumber(n)
```

> returns the catalan number for the integer argument `n`.
 
See:  
* [Wikipedia - Catalan number](http://en.wikipedia.org/wiki/Catalan_number)

### Examples

```
>> CatalanNumber(4)
14
```
