## Roots

```
Roots(polynomial-equation, var)
```

> determine the roots of a univariate polynomial equation with respect to the variable `var`.

### Examples

```
>> Roots(3*x^3-5*x^2+5*x-2==0,x)
x==2/3||x==1/2-I*1/2*Sqrt(3)||x==1/2+I*1/2*Sqrt(3)
```
