## IntegerPart

```
IntegerPart(expr)
```

> for real `expr` return the integer part of `expr`.
 
See
* [Wikipedia - Floor and ceiling functions](https://en.wikipedia.org/wiki/Floor_and_ceiling_functions)

### Examples

```
>> IntegerPart(2.4)
2

>> IntegerPart(-9/4)
-2
```
