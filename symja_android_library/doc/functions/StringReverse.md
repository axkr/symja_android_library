## StringReverse

```
StringReplace(string)
```

> reverse the `string`.

See
* [Wikipedia - Palindrome](https://en.wikipedia.org/wiki/Palindrome)

### Examples

A famous palindrome:

```
>> StringReverse("Never odd or even")
neve ro ddo reveN
```
