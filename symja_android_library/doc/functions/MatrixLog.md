## MatrixLog

```
MatrixLog(matrix)
```

> computes the matrix logarithm of the square `matrix`.
 
See
* [Wikipedia - Logarithm of a matrix](https://en.wikipedia.org/wiki/Logarithm_of_a_matrix)
* [Wikipedia - Matrix exponential](https://en.wikipedia.org/wiki/Matrix_exponential)

### Examples

```
>> MatrixLog({{3.4, 1.2}, {0.001, -0.9}}) 
{{1.22377+I*0.00020385,0.37081+I*(-0.87661)},
 {0.000309008+I*(-0.000730508),-0.104964+I*3.14139}}
```
