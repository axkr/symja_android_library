## BitSet

```
BitSet(i, b)
```

> set the `b`th bit in the integer `i`.  
 
### Examples

``` 
>> IntegerDigits(32,2) 
{1,0,0,0,0,0}

>> BitSet(32,1)
34 

>> IntegerDigits(34,2)
{1,0,0,0,1,0}
```
