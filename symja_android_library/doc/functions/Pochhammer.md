## Pochhammer

```
Pochhammer(a, n)
```

> returns the pochhammer symbol for a rational number `a` and an integer number `n`.

See
* [Wikipedia - Falling and rising factorials](https://en.wikipedia.org/wiki/Falling_and_rising_factorials)

### Examples

```
>> Pochhammer(4, 8)
6652800
```
