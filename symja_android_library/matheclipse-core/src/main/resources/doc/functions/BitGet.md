## BitGet

```
BitGet(i, n)
```

> gets the `n`th bit in the integer `i`. 
 
### Examples

``` 
>> IntegerDigits(36,2)
{1,0,0,1,0,0}

>> BitGet(36,2)
1

>> BitGet(36,-4)
BitGet(36,-4)
```
