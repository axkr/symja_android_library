Adaptors to MAS classes.

This package is deprecated and will be removed. It contains classes with the MAS module and method names. Semantics may not be preserved but should work with some preprocessing code transformations. For example ADV(a,b,c) must be substituted by b=FIRST(a); c=RED(a).
