## SinIntegral

```
SinIntegral(expr)
```

> returns the hyperbolic sine integral of `expr`.
  
See
* [Wikipedia - Trigonometric integral](https://en.wikipedia.org/wiki/Trigonometric_integral)

### Examples

```
>> SinIntegral(0)
0 
```
