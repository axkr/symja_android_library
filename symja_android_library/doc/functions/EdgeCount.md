## EdgeCount

``` 
EdgeCount(graph)
```

> return the number of edges of the `graph`

See:
* [Wikipedia - Graph theory](https://en.wikipedia.org/wiki/Graph_theory)

### Examples

```
>> EdgeCount(CompleteGraph(4))
6
```
