## Cofactor

```
Cofactor(matrix,  {i,j})
```

> calculate the cofactor of the matrix


### Examples

```
>> Cofactor({{6, 0, 4, 9, 5}, {1, 9, 3, 1, 2}, {5, 4, 5, 3, 8}, {3, 9, 8, 2, 5}, {4, 1, 6, 6, 4}},{1,3})
-30
```
