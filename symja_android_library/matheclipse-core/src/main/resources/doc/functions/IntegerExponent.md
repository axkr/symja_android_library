## IntegerExponent

```
IntegerExponent(n, b)
```

> gives the highest exponent of `b` that divides `n`.

### Examples

```
>> IntegerExponent(16, 2)
4

>> IntegerExponent(-510000)
4

>> IntegerExponent(10, b)
IntegerExponent(10, b)
```
