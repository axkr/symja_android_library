## NRoots

```
NRoots(polynomial==0)
```

> gives the numerical roots of a univariate polynomial `polynomial`.

### Examples

```
>> NRoots(x^3-4*x^2+x+6==0)
{2.9999999999999996,-1.0000000000000002,1.9999999999999998}
```
