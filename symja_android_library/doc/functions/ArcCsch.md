## ArcCsch

```
ArcCsch(z)
```

> returns the inverse hyperbolic cosecant of `z`.

See:
* [Wikipedia: Inverse hyperbolic function](https://en.wikipedia.org/wiki/Inverse_hyperbolic_function)

### Examples

``` 
>> ArcCsch(0)    
ComplexInfinity 
  
>> ArcCsch(1.0)    
0.8813735870195429
```
