## SymmetricMatrixQ

```
SymmetricMatrixQ(m)
```

> returns `True` if `m` is a symmetric matrix.

See
* [Wikipedia - Symmetric matrix](https://en.wikipedia.org/wiki/Symmetric_matrix)


### Examples

```
>> SymmetricMatrixQ({{1,7,3}, {7,4,-5}, {3,-5,6}})
True 
```
