## ExpIntegralE

```
ExpIntegralE(n, expr)
```

> returns the exponential integral `E_n(expr)` of `expr`.
  
See
* [Wikipedia - Exponential integral](https://en.wikipedia.org/wiki/Exponential_integral)

### Examples

```
>> ExpIntegralE(0, z)
1/(E^z*z) 
```
