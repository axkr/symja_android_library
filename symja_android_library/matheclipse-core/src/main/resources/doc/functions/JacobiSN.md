## JacobiSN

```
JacobiSN(x, m)
```

> returns the Jacobian elliptic function `sn(x, m)`. 
   

See
* [Wikipedia - Jacobi elliptic functions](https://en.wikipedia.org/wiki/Jacobi_elliptic_functions)
* [NIST - Jacobian elliptic functions](https://dlmf.nist.gov/22.5)

### Examples

```
>> JacobiSN(10.0,1/3)
0.389397
```
