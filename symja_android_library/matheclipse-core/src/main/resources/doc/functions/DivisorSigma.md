## DivisorSigma

```
DivisorSigma(k, n)
```

>  returns the sum of the `k`-th powers of the divisors of `n`.

See:  
* [Wikipedia - Divisor function](https://en.wikipedia.org/wiki/Divisor_function)

## Examples

```
>> DivisorSigma(0,12)
6

>> DivisorSigma(1,12)
28
```
