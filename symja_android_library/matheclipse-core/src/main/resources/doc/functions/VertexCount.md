## EdgeCount

``` 
VertexCount(graph)
```

> return the number of vertices of the `graph`

See:
* [Wikipedia - Graph theory](https://en.wikipedia.org/wiki/Graph_theory)

### Examples

```
>> VertexCount(CompleteGraph(4))
4
```
