## MaxFilter

```
MaxFilter(list, r)
```

> filter which evaluates the `Max` of `list` for the radius `r`. 

### Examples
 
```
>> MaxFilter({1, 2, 3, 2, 1}, 1) 
{2,3,3,3,2}
```
