## Less

```
Less(x, y) 

x < y
```

> yields `True` if `x` is known to be less than `y`.

```
lhs < rhs
```

> represents the inequality `lhs < rhs`.
 
	
### Examples
 
```
>> 3<4
True

>> {Less(), Less(x), Less(1)}
{True, True, True}
```
