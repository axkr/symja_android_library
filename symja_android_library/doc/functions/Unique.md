## Unique

```
Unique(expr)
```
 
> create a unique symbol of the form `expr$...`.

```
Unique("expr")
```
 
> create a unique symbol of the form `expr...`.

### Examples
 
```
>> Unique(xy)
xy$1

>> Unique("a")
a1
```
