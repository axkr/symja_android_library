## Not

```
Not(expr)
```

or

```
!expr
```

> Logical Not function (negation). Returns `True` if the statement is `False`. Returns `False` if the `expr` is `True`

See
* [Wikipedia - Negation](https://en.wikipedia.org/wiki/Negation)

### Examples

```
>> !True
False
>> !False
True
>> !b
!b
```
