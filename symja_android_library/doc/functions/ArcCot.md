## ArcCot

```
ArcCot(z)
```

> returns the inverse cotangent of `z`.

See:
* [Wikipedia - Inverse trigonometric functions](https://en.wikipedia.org/wiki/Inverse_trigonometric_functions)

### Examples

``` 
>> ArcCot(0)    
Pi/2  
  
>> ArcCot(1)    
Pi/4
```
