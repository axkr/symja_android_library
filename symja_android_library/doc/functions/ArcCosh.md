## ArcCosh

```
ArcCosh(z)
```

> returns the inverse hyperbolic cosine of `z`.

See:
* [Wikipedia: Inverse hyperbolic function](https://en.wikipedia.org/wiki/Inverse_hyperbolic_function)

### Examples

```
>> ArcCosh(0)
I*1/2*Pi

>> ArcCosh(0.0)
I*1.5707963267948966

>> ArcCosh(1.4)
0.867014726490565
```
