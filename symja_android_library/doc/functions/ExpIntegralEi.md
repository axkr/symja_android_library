## ExpIntegralEi

```
ExpIntegralEi(expr)
```

> returns the exponential integral `Ei(expr)` of `expr`.
  
See
* [Wikipedia - Exponential integral](https://en.wikipedia.org/wiki/Exponential_integral)

### Examples

```
>> ExpIntegralEi(0)
-Infinity
```
