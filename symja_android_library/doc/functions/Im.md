## Im

```
Im(z)
```

> returns the imaginary component of the complex number `z`.
 
See
* [Wikipedia - Complex number](https://en.wikipedia.org/wiki/Complex_number)

### Examples

```
>> Im(3+4*I)
4

>> Im(0.5 + 2.3*I)
2.3
```
