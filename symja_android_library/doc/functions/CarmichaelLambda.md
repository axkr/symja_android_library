## CarmichaelLambda

```
CarmichaelLambda(n)
```

>  the Carmichael function of `n`
 
See:  
* [Wikipedia - Carmichael function](https://en.wikipedia.org/wiki/Carmichael_function)

## Examples

```
>> CarmichaelLambda(35)
12
```
