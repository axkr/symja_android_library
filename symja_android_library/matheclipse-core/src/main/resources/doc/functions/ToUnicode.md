## ToUnicode

```
ToUnicode(string)
```

> converts `string` into a string of corresponding unicode character codes.

 
### Examples

```
>> ToUnicode("123abcABC")
"\u0031\u0032\u0033\u0061\u0062\u0063\u0041\u0042\u0043"
```
