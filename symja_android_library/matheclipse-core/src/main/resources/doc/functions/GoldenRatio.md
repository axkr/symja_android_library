## GoldenRatio

```
GoldenRatio
```

> is the golden ratio `(1+Sqrt(5))/2`.

See
* [Wikipedia - Golden ratio](https://en.wikipedia.org/wiki/Golden_ratio)
* [Fungrim - Golden ratio](http://fungrim.org/topic/Golden_ratio/)
 
### Examples

``` 
>> N(GoldenRatio)
1.618033988749895
```
