## Histogram 

```
Histogram(list-of-values)
```

> plots a histogram for a `list-of-values`

### Examples
 
```
>> Histogram({1, 2, 3, None, 3, 5, f(), 2, 1, 5,4,3,2,foo, 2, 3})
```
