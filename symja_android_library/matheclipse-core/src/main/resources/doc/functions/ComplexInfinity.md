## ComplexInfinity

```
ComplexInfinity
```

> represents an infinite complex quantity of undetermined direction.

See 
* [Wikipedia - Infinity](https://en.wikipedia.org/wiki/Infinity) 

### Examples

```
>> 1 / ComplexInfinity
0
 
>> ComplexInfinity + ComplexInfinity
ComplexInfinity
 
>> ComplexInfinity * Infinity
ComplexInfinity
 
>> FullForm(ComplexInfinity)
DirectedInfinity()
```
