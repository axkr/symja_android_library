## AssociationQ

```
AssociationQ(expr) 
```

> returns `True` if `expr` is an association, and `False` otherwise.
 
### Examples

```
>> AssociationQ(<|ahey->avalue, bkey->bvalue, ckey->cvalue|>)
True
```
