## InverseHaversine

```
InverseHaversine(z)
```

> returns the inverse haversine function of `z`.


See
* [Wikipedia - Versine](https://en.wikipedia.org/wiki/Versine#ahav)


### Examples
```
>> InverseHaversine(0.5)
1.5707963267948968 

>> InverseHaversine(1 + 2.5 * I)
1.764589463349828+I*2.3309746530493127 
```
