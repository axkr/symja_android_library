## RandomGraph

```
RandomGraph({number-of-vertices,number-of-edges})
```

> create a random graph with `number-of-vertices` vertices and `number-of-edges` edges. 

### Examples

```
>> RandomGraph({5,10})
Graph({1,2,3,4,5},{5<->3,4<->2,5<->1,2<->3,5<->4,2<->5,2<->1,3<->4,3<->1,1<->4})
```
