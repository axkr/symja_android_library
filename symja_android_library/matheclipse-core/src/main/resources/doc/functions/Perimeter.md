## Perimeter

```
Perimeter(geometric-form)
```

> returns the perimeter of the `geometric-form`.
  

See:
* [Wikipedia - Perimeter](https://en.wikipedia.org/wiki/Perimeter)
 

### Examples

```
>> Perimeter(Disk({1,2}))
2*Pi
```
